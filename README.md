# Sephara IA | Proposta Comercial

Site em React + Vite + Tailwind + TypeScript para a proposta comercial do sistema inteligente de controle de distribuição de merenda escolar por biometria.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy na Vercel

Ao importar este projeto na Vercel, use as configurações padrão de Vite:

- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`
